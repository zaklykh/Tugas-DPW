Jika anda mengklik tombol "Kirim", data yang dimasukkan akan dikirim ke server sesuai dengan URL yang ditentukan dalam atribut "action" pada tag form. Jika anda mengklik tombol "Reset", semua input dalam form akan dikembalikan ke nilai awalnya.

Form berfungsi untuk mengumpulkan data dari pengguna dan mengirimkannya ke server untuk diproses. Form terdiri dari elemen-elemen seperti input, textarea, select, dan button yang memungkinkan pengguna untuk memasukkan data. Atribut action pada tag form menentukan URL tempat data akan dikirimkan saat form disubmit, sedangkan atribut method menentukan metode HTTP yang digunakan untuk mengirim data (biasanya "GET" atau "POST").
