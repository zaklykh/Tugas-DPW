label berfugnsi sebagai pemberi nama atau keterangan untuk input yang ada

input berfungsi sebagai komponen utama untuk menerima data dari pengguna. Perilakunya berubah bergantung pada atribut

type="text" berfungsi untuk input teks standar
type="password" berfungsi untuk input password yang akan dienkripsi
type="radio" berfungsi untuk input pilihan yang hanya bisa dipilih satu dari beberapa opsi

Uji coba 3



type "checkbox" berfungsi untuk input pilihan yang bisa dipilih lebih dari satu opsi

Uji coba 4






type "color" berfungsi untuk input pilihan warna dengan menampilkan palet warna
type "date" berfungsi untuk input tanggal dengan menampilkan kalender
type "button" berfungsi untuk membuat tombol yang dapat diklik untuk menjalankan fungsi JavaScript tertentu

Uji coba 5

type "email" berfungsi untuk input alamat email dengan validasi format email
