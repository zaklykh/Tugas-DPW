Nomor 8: Uraian AJAX dan jQuery

1. Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah teknik pengembangan web yang memungkinkan halaman web memperbarui data secara asynchronous (tidak sinkron). Ini berarti halaman dapat mengirim dan menerima data dari server di latar belakang tanpa harus memuat ulang (refresh) seluruh halaman web tersebut.

2. Apa itu jQuery?

jQuery adalah library JavaScript yang cepat, kecil, dan kaya fitur. jQuery dirancang untuk menyederhanakan penulisan skrip sisi klien (client-side script), seperti manipulasi dokumen HTML, penanganan event, animasi, dan mempermudah penggunaan AJAX

3. Cara Menggunakan jQuery

Untuk menggunakan jQuery, Anda perlu memasukkan library jQuery ke dalam file HTML Anda, baik dengan mengunduh file-nya secara lokal atau menggunakan Content Delivery Network (CDN) seperti yang dilakukan pada kode ini.
