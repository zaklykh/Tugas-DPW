Daftar Tugas Modul 4: CSS Dasar

Silakan klik tautan di bawah ini untuk melihat hasil praktikum:

  • 1a. Simple Selectors
    Latihan penggunaan Elemen, ID (#), dan Class (.) selector.
  • 1b. Combinator Selectors
    Latihan Descendant, Child, Adjacent, dan General Sibling selectors.
  • 1c. Pseudo-class Selectors
    Mengatur gaya elemen saat interaksi (Hover pada Div dan Link).
  • 1d. Pseudo-elements Selectors
    Menggunakan ::first-line untuk dekorasi teks artikel.
  • 2. External Style Sheet (Dasar)
    Menghubungkan HTML dengan file style1.css.
  • 3. Selector ID (External)
    Implementasi spesifik ID dan Class menggunakan style2.css.
  • 4. Border Style
    Eksperimen berbagai jenis bingkai (Solid, Dotted, Ridge, dll).
  • 5. Pengaturan Margin
    Mengatur jarak luar elemen (Atas, Kanan, Bawah, Kiri).
  • 7. Pengaturan Padding
    Mengatur jarak dalam antara konten dan border.
  • 6. Hyperlink Advanced
    Custom link dengan Text Transform dan Decoration.
  • 8. Display Property
    Mengubah elemen Inline (span) menjadi Block.
  • 9. Studi Kasus: Laman Web Lengkap
    Antarmuka web Politeknik Negeri Madiun menggunakan struktur Layout lengkap.
